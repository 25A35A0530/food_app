# Food Ordering Web Application

This is a full-stack Food Ordering Web Application developed as part of my learning and internship tasks. The application allows users to browse food items, add them to cart, place orders, and track their order status in real-time. It also includes an admin panel to manage and update orders efficiently.

## Features

### User Features

* Browse food menu with search functionality 
* Add items to cart 
* Place orders easily
* Track order status (Placed → Preparing → Delivered )
* View detailed order history

### Admin Features

* View all user orders
* Update order status (Preparing / Delivered)
* Dashboard with key insights:

  * Total Orders 
  * Total Revenue 
  * Delivered Orders

## How to Run the Project

### Backend Setup

1. Move the project to:
2. Start **Apache** and **MySQL** in XAMPP
3. Import your database using phpMyAdmin
4. Make sure backend APIs are running

### Frontend Setup

1. Navigate to frontend folder: cd frontend
2. Install dependencies: npm install
3. Start the React app: npm start
